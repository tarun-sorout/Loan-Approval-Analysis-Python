# 💰 Loan Approval Analysis

This project focuses on analyzing loan application data using Python. The objective is to understand the factors that influence loan approval decisions through data cleaning, exploratory data analysis (EDA), and visualization techniques.

The project demonstrates how financial institutions can use data analysis to identify approval patterns and gain meaningful insights from customer loan applications.

---

## 📌 Project Overview

The main goal of this project is to analyze loan application records and discover the key factors that affect loan approval status.

The notebook includes:

- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Missing Value Handling
- Loan Approval Analysis
- Customer Profile Analysis
- Data Visualization
- Business Insights
